# Checklist Audit v1.0

Gunakan checklist ini sebelum aplikasi dipakai untuk kebutuhan internal lembaga.

## A. Login dan Role

- [ ] Admin bisa login.
- [ ] Guru bisa login.
- [ ] Bendahara bisa login.
- [ ] User tanpa profile mendapat pesan jelas.
- [ ] Admin bisa melihat semua menu.
- [ ] Guru tidak melihat SPP dan Pengaturan.
- [ ] Bendahara tidak melihat Nilai, Raport, Guru, Absensi, dan Pengaturan.

## B. Data Santri

- [ ] Tambah santri.
- [ ] Edit santri.
- [ ] Detail santri.
- [ ] Hapus data percobaan.
- [ ] Search/filter berjalan jika ada.

## C. Nilai dan Raport

- [ ] Input nilai.
- [ ] Simpan nilai.
- [ ] Edit nilai.
- [ ] Tidak duplikat.
- [ ] Preview raport.
- [ ] Print raport A4.

## D. Absensi Guru

- [ ] Tambah guru.
- [ ] Input absensi.
- [ ] Edit absensi.
- [ ] Tidak duplikat per tanggal.

## E. SPP

- [ ] Buat tagihan.
- [ ] Input pembayaran sebagian.
- [ ] Input pembayaran lunas.
- [ ] Status berubah benar.
- [ ] Rekap tunggakan benar.
- [ ] Export CSV berjalan.
- [ ] Pengingat WhatsApp membuka pesan yang benar.

## F. Deploy

- [ ] `npm run build` berhasil.
- [ ] Vercel env sudah diisi.
- [ ] Production route bisa dibuka.
- [ ] Login production berjalan.
- [ ] Supabase production berjalan.
